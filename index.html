About This Site

Lost Star is a lightweight static web app that comes with two core features:

  • An AI assistant which can answer questions using a variety of public data sources (Wikipedia, Reddit, etc.) even when no API key is configured.
  • A bookmark manager tuned for manga/manhwa that lets you search, save, and categorize links directly in your browser.

All data is stored locally in the browser, and the AI will try to help you with common queries, recommendations, or information lookup.
